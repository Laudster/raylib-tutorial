Første Program

For å lage et prosjekt for visual studio code kan du bare kopiere prosjektet innenfor C:\raylib\raylib\projects kalt VSCODE. Du kan plassere det hvor som helst og det bør fungere ved å presse f5.

Prosjektet du finner innenfor der er imidlertid i 3d, men i begynnelsen vil jeg anbefale å bare forholde deg til 2d. Så her er et lett eksempel for 2d. 

#include "raylib.h"

int main()
{
    InitWindow(1200, 800, "game");
    SetTargetFPS(60);

    while (!WindowShouldClose())
    {
        BeginDrawing();
            ClearBackground(RAYWHITE);
            DrawText("Raylib fungerer", 220, 300, 100, BLACK);
        EndDrawing();
    }
    CloseWindow();
    return 0;
}
